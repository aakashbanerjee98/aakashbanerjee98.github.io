Sentiment Analysis of Regional Languages

This project aims to analyze the opinionated texts which are present in social media and create a model which can correctly predict the sentiment of future texts.This model is then used to create a web app that can analyze the sentiment of any bengali text that has been submitted by the user.This web app can then be used to analyze various reviews that were written in bengali languages which were posted on different social medias and other forums.
